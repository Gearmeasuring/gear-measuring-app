# 齿轮波纹度分析算法说明

## 一、算法概述

波纹度分析用于评估齿轮齿面表面的周期性波动特征，通过频谱分析方法提取不同频率的波纹成分。

## 二、核心算法流程

### 1. 数据预处理

#### 1.1 齿形数据处理
```
对于每个齿的测量数据：
1. 去除鼓形（二元二次多项式拟合）
2. 去除斜率偏差（一元一次多项式拟合）
3. 计算渐开线极角
```

**渐开线极角计算**:
```
inv(α) = tan(α) - α
其中 α = arccos(rb/r)
rb = 基圆半径
r = 测量点半径
```

#### 1.2 齿向数据处理
```
轴向旋转角度 Δφ = 2 × Δz × tan(β₀) / D₀

其中:
- Δz = 相对于评价范围中心的轴向距离
- β₀ = 节圆处的螺旋角
- D₀ = 节圆直径
```

### 2. 角度合成（标准公式）

**根据标准公式**:
```
φ = -ξ - Δφ + τ
```

其中:
- **φ**: 旋转角度
- **ξ (xi)**: 滚动角（渐开线极角）
- **Δφ (delta_phi)**: 轴向位置产生的旋转
- **τ (tau)**: 节距角 = 齿序号 × 360° / 齿数

#### 2.1 齿形（渐开线）
对于齿形，轴向旋转 Δφ = 0:
```
φ = -ξ + τ
```
- **ξ**: 渐开线极角 `inv(α) = tan(α) - α`
- **τ**: 节距角

#### 2.2 齿向（螺旋线）
对于齿向，滚动角 ξ = 0:
```
φ = -Δφ + τ
```
- **Δφ**: 轴向位置旋转 `2 × Δz × tan(β₀) / D₀`
  - Δz = 相对于评价范围中心的轴向距离
  - β₀ = 节圆处的螺旋角
  - D₀ = 节圆直径
- **τ**: 节距角 = 齿序号 × 360° / 齿数

#### 2.3 归一化
```
所有角度归一化到 0-360 度范围
```

### 3. 频谱分析

#### 3.1 数据插值
```
1. 去除重复角度点
2. 在0-360度范围内均匀插值（点数 = max(360, 2×5×ZE+10)）
3. 满足奈奎斯特采样定理
```

#### 3.2 直接最小二乘法频谱计算
```
对于每个目标阶次 order:
    1. 构建矩阵 A = [cos(order×θ), sin(order×θ)]
    2. 求解 Ax = b，得到系数 [A, B]
    3. 振幅 = √(A² + B²) × 缩放因子
```

**最小二乘法拟合**:
```
构建矩阵 A = [cos(order×θ), sin(order×θ)]
求解 Ax = b，得到系数 [A, B]
振幅 = √(A² + B²)
相位 = arctan2(A, B)
```

### 4. 振幅缩放因子

根据实测数据分析，振幅需要乘以缩放因子 **0.1** 才能匹配Klingelnberg参考值。

### 5. 高阶波纹度计算

#### 5.1 高阶分量筛选
```
高阶分量 = 波数 ≥ ZE（齿数）的所有分量
```

#### 5.2 评价指标
```
W值（高阶总振幅）= Σ(高阶分量振幅)
RMS值 = √(mean(高阶重构信号²))
```

## 三、算法参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| ZE | 齿数 | 从齿轮参数获取 |
| 最大搜索阶次 | 5×ZE | 435（当ZE=87时） |
| 插值点数 | max(360, 2×5×ZE+10) | 880 |
| 振幅缩放因子 | 0.1 | 0.1 |

## 四、输出结果

### 4.1 频谱信息
- 目标阶次的波数、振幅、相位
- 高阶分量列表（波数≥ZE）

### 4.2 评价指标
- W值：高阶总振幅（μm）
- RMS值：高阶均方根值（μm）

### 4.3 图表
1. 目标阶次振幅柱状图
2. 原始信号vs重构信号
3. 残差信号
4. 高阶分量合成

## 五、算法特点

1. **最小二乘法**：使用最小二乘法拟合正弦波，精度高
2. **直接计算**：直接计算所有目标阶次，避免迭代分解遗漏
3. **振幅缩放**：使用固定缩放因子0.1
4. **全齿分析**：合并所有齿的数据，得到完整的0-360度频谱

## 六、应用场景

- 齿轮加工质量评估
- 磨齿机精度检测
- 齿轮噪声预测
- 加工工艺优化

## 七、验证结果

### 样本1 (ZE=87)
- 平均误差: 87.4%

### 样本2 (ZE=26)
- 平均误差: 78.1%
- 右齿形最佳缩放因子: 0.1083，误差: 22.5%

### 关键发现
1. 数据单位：MKA文件数据为微米(μm)
2. 振幅缩放因子约0.1（接近1/√ZE）
3. 合并曲线法比每个齿单独计算频谱法效果更好
4. 直接最小二乘法可以提取所有目标阶次
