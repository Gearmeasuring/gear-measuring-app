# 一键部署指南

## 方案一：使用 GitHub Desktop（推荐，最简单）

### 步骤 1：安装 GitHub Desktop
1. 下载：https://desktop.github.com/
2. 安装并登录你的 GitHub 账号

### 步骤 2：创建仓库
1. 打开 GitHub Desktop
2. 点击 "File" → "New repository"
3. 填写信息：
   - Name: `gear-measuring-app`
   - Local path: 选择项目目录的父文件夹
   - 勾选 "Initialize this repository with a README"
4. 点击 "Create repository"

### 步骤 3：复制文件
将项目文件复制到新创建的仓库目录中。

### 步骤 4：提交并推送
1. 在 GitHub Desktop 中，你会看到所有更改
2. 填写提交信息："Initial commit"
3. 点击 "Commit to main"
4. 点击 "Publish repository"
5. 选择 "Public" 或 "Private"
6. 点击 "Publish repository"

### 步骤 5：部署到 Streamlit Cloud
1. 访问 https://streamlit.io/cloud
2. 点击 "Sign in with GitHub"
3. 点击 "New app"
4. 选择 `gear-measuring-app` 仓库
5. 设置主文件路径：`web_app/app_professional.py`
6. 点击 "Deploy"

## 方案二：手动命令行部署

### 步骤 1：安装 Git
下载：https://git-scm.com/download/win

### 步骤 2：配置 Git
```bash
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub邮箱"
```

### 步骤 3：初始化并推送
在项目目录中打开命令提示符：

```bash
# 进入项目目录
cd "e:\python\gear measuring software - 20251217\gear measuring software - 20251217backup"

# 初始化 Git 仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 齿轮测量报告系统"
```

### 步骤 4：在 GitHub 创建仓库
1. 访问 https://github.com/new
2. 填写 Repository name: `gear-measuring-app`
3. **不要**勾选 "Initialize this repository with a README"
4. 点击 "Create repository"
5. 复制仓库地址（如：`https://github.com/你的用户名/gear-measuring-app.git`）

### 步骤 5：关联并推送
```bash
# 关联远程仓库（替换为你的地址）
git remote add origin https://github.com/你的用户名/gear-measuring-app.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

### 步骤 6：部署到 Streamlit Cloud
1. 访问 https://streamlit.io/cloud
2. 用 GitHub 登录
3. 点击 "New app"
4. 选择 `gear-measuring-app` 仓库
5. 设置主文件路径：`web_app/app_professional.py`
6. 点击 "Deploy"

## 方案三：使用 VS Code（如果你用 VS Code）

### 步骤 1：安装 Git
下载：https://git-scm.com/download/win

### 步骤 2：在 VS Code 中初始化
1. 打开项目文件夹
2. 点击左侧的 "源代码管理" 图标（或按 Ctrl+Shift+G）
3. 点击 "初始化仓库"
4. 输入提交信息："Initial commit"
5. 点击提交按钮

### 步骤 3：推送到 GitHub
1. 按 Ctrl+Shift+P 打开命令面板
2. 输入 "Git: Add Remote"
3. 输入远程名称：origin
4. 输入 GitHub 仓库地址
5. 点击 "发布分支"

### 步骤 4：部署
同方案一的步骤 5

## 更新代码

修改代码后，需要重新推送：

```bash
git add .
git commit -m "更新说明"
git push origin main
```

Streamlit Cloud 会自动重新部署。

## 常见问题

### Q: 推送失败怎么办？
A: 先拉取最新代码：
```bash
git pull origin main
git push origin main
```

### Q: 如何设置私有仓库？
A: 在 GitHub 创建仓库时选择 "Private"

### Q: Streamlit Cloud 免费吗？
A: 是的，有免费额度，足够个人使用

### Q: 国内访问慢怎么办？
A: 可以考虑使用阿里云函数计算或腾讯云部署
