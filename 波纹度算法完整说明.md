4. 齿形极角计算算法
================================================================================

4.1 渐开线理论基础
-------------------
渐开线方程:
    极角 θ = inv(α) = tan(α) - α
    
其中压力角 α 由以下关系确定:
    cos(α) = rb / r
    
    rb = 基圆半径
    r = 测量点半径

4.2 展长角度计算
-----------------
展长 s(d): 从基圆到测量点的渐开线弧长
    s(d) = sqrt((d/2)² - (db/2)²)
    
展长对应的旋转角 ξ:
    ξ = s / (π × db) × 360°

4.3 相对极角计算
-----------------
从齿顶开始排列，齿顶极角为0°:
    相对极角 = 展长角度 - 齿顶展长角度
    
    φ_profile(d) = ξ(d) - ξ(d_tip)
    
其中 d_tip = 终评点直径（齿顶方向）

4.4 旋转角度合成
-----------------
对于第n个齿（齿号从1开始）:
    齿序角 τ = (n-1) × 节距角 = (n-1) × 360° / ZE

右齿形:
    最终角度 = τ + φ_profile
    
左齿形:
    最终角度 = τ - φ_profile

================================================================================
5. 齿向极角计算算法
================================================================================

5.1 螺旋角产生的旋转
---------------------
当测量点沿齿向移动时，由于螺旋角的存在，会产生额外的旋转角度。

5.2 极角计算公式
-----------------
    φ_helix = 2 × Δz × tan(β) / D₀
    
其中:
    Δz = 测量点位置 - 起评点位置
    β = 螺旋角（绝对值）
    D₀ = 节圆直径

转换为度数:
    φ_helix_deg = degrees(φ_helix)

5.3 旋转角度合成
-----------------
对于第n个齿:
    齿序角 τ = (n-1) × 节距角

右齿向:
    最终角度 = τ + φ_helix
    
左齿向:
    最终角度 = τ - φ_helix

6. 数据预处理（鼓形和斜率剔除）
================================================================================

6.1 目的
---------
消除测量数据中的系统偏差:
- 鼓形: 齿面中部的凸起或凹陷（二次曲线特征）
- 斜率: 整体倾斜趋势（线性特征）

6.2 算法步骤
-------------
步骤1: 二次多项式拟合（去除鼓形）
    y = ax² + bx + c
    
    使用最小二乘法拟合系数 [a, b, c]
    
    去除鼓形: data_no_crown = data - (ax² + bx + c)

步骤2: 线性拟合（去除斜率）
    y = kx + d
    
    使用最小二乘法拟合系数 [k, d]
    
    去除斜率: data_corrected = data_no_crown - (kx + d)

6.3 实现代码
-------------
```python
def remove_crown_and_slope(data: np.ndarray) -> np.ndarray:
    if len(data) < 3:
        return np.array(data, dtype=float)
    
    n = len(data)
    x = np.linspace(-1, 1, n)
    
    # 步骤1: 去除鼓形（二次多项式）
    A_crown = np.column_stack((x**2, x, np.ones(n)))
    coeffs_crown, _, _, _ = np.linalg.lstsq(A_crown, data, rcond=None)
    a, b, c = coeffs_crown
    crown = a * x**2 + b * x + c
    data_no_crown = data - crown
    
    # 步骤2: 去除斜率（线性）
    A_slope = np.column_stack((x, np.ones(n)))
    coeffs_slope, _, _, _ = np.linalg.lstsq(A_slope, data_no_crown, rcond=None)
    k, d = coeffs_slope
    slope = k * x + d
    data_corrected = data_no_crown - slope
    
    return data_corrected
```

步骤2: 按角度排序

步骤3: 去除重复点
    unique_angles, unique_indices = np.unique(np.round(angles, 3), return_index=True)
    unique_values = values[unique_indices]

步骤4: 插值到均匀网格
    num_interp = max(720, 4 * teeth_count)  # 至少720点或4倍齿数
    interp_angles = np.linspace(0, 360, num_interp)
    interp_values = np.interp(interp_angles, unique_angles, unique_values, period=360)

7.3 实现代码
-------------
```python
def build_closed_curve(angles: np.ndarray, values: np.ndarray, 
                       teeth_count: int) -> Tuple[np.ndarray, np.ndarray]:
    # 归一化到0-360度
    angles = angles % 360.0
    angles[angles < 0] += 360.0
    
    # 排序
    sort_idx = np.argsort(angles)
    angles = angles[sort_idx]
    values = values[sort_idx]
    
    # 去重
    unique_angles, unique_indices = np.unique(np.round(angles, 3), return_index=True)
    unique_values = values[unique_indices]
    
    # 插值
    num_interp = max(720, 4 * teeth_count)
    interp_angles = np.linspace(0, 360, num_interp)
    interp_values = np.interp(interp_angles, unique_angles, unique_values, period=360)
    
    return interp_angles, interp_values
```

8. 频谱分析算法（最小二乘法迭代分解）
================================================================================

8.1 正弦波模型
---------------
对于阶次为k的正弦波:
    y = A × cos(k×θ) + B × sin(k×θ)
    
等价于:
    y = C × sin(k×θ + φ)
    
其中:
    C = √(A² + B²)  (振幅)
    φ = arctan2(A, B)  (相位)

8.2 最小二乘法拟合
-------------------
构建设计矩阵:
    X = [cos(k×θ), sin(k×θ)]
    
求解:
    [A, B]ᵀ = (XᵀX)⁻¹Xᵀy
    
使用numpy:
    coeffs, _, _, _ = np.linalg.lstsq(X, values, rcond=None)
    A, B = coeffs[0], coeffs[1]

8.3 迭代分解算法
-----------------
算法流程:
```
输入: angles (角度数组), values (偏差值数组), num_components (提取分量数)

输出: components (频谱分量列表)

步骤:
1. 初始化:
   - angles_rad = radians(angles)
   - residual = values - mean(values)
   - components = []
   - extracted_orders = set()

2. 迭代提取 (i = 1 to num_components):
   a. 在所有未提取的阶次中搜索:
      for order in 1 to max_order:
          if order not in extracted_orders:
              拟合正弦波，计算振幅
   
   b. 找到振幅最大的阶次 best_order
   
   c. 记录频谱分量:
      components.append({
          order: best_order,
          amplitude: best_amplitude,
          phase: best_phase,
          coefficient_a: A,
          coefficient_b: B
      })
   
   d. 从残差中移除该分量:
      fitted = A × cos(best_order × angles_rad) + B × sin(best_order × angles_rad)
      residual = residual - fitted
   
   e. 标记已提取:
      extracted_orders.add(best_order)

3. 返回 components
```

8.4 实现代码
-------------
```python
def fit_sine_wave(angles_rad: np.ndarray, values: np.ndarray, 
                  order: int) -> SpectrumComponent:
    cos_term = np.cos(order * angles_rad)
    sin_term = np.sin(order * angles_rad)
    
    A = np.column_stack([cos_term, sin_term])
    coeffs, _, _, _ = np.linalg.lstsq(A, values, rcond=None)
    
    a, b = coeffs[0], coeffs[1]
    amplitude = np.sqrt(a**2 + b**2)
    phase = np.arctan2(a, b)
    
    return SpectrumComponent(
        order=order,
        amplitude=amplitude,
        phase=phase,
        coefficient_a=a,
        coefficient_b=b
    )

def iterative_decomposition(angles: np.ndarray, values: np.ndarray,
                            num_components: int = 10,
                            max_order: int = None,
                            teeth_count: int = 87) -> List[SpectrumComponent]:
    if max_order is None:
        max_order = 5 * teeth_count
    
    angles_rad = np.radians(angles)
    residual = np.array(values, dtype=float)
    residual = residual - np.mean(residual)
    
    components = []
    extracted_orders = set()
    
    for i in range(num_components):
        # 搜索最大振幅阶次
        best_order = 1
        best_amplitude = 0
        best_component = None
        
        for order in range(1, max_order + 1):
            if order in extracted_orders:
                continue
            
            component = fit_sine_wave(angles_rad, residual, order)
            
            if component.amplitude > best_amplitude:
                best_amplitude = component.amplitude
                best_order = order
                best_component = component
        
        # 记录并移除
        extracted_orders.add(best_order)
        components.append(best_component)
        
        fitted = best_component.coefficient_a * np.cos(best_order * angles_rad) + \
                 best_component.coefficient_b * np.sin(best_order * angles_rad)
        residual = residual - fitted
    
    return components
```

9. 高阶波纹度评价
================================================================================

9.1 定义
---------
高阶波纹度: 阶次 ≥ 齿数ZE的波纹度分量

评价方式: 0到360度之间有多少个波，一周中有多少个波。
例如: 87个波 → 阶次/频率 = 87

9.2 计算方法
-------------
步骤1: 筛选高阶分量
    high_order_components = [c for c in components if c.order >= ZE]

步骤2: 计算总振幅
    W = Σ(amplitude_i) for all high-order components

步骤3: 重构高阶信号
    reconstructed = Σ(A_i × cos(order_i × θ) + B_i × sin(order_i × θ))

步骤4: 计算RMS
    RMS = √(mean(reconstructed²))

9.3 实现代码
-------------
```python
def evaluate_high_order(components: List[SpectrumComponent],
                        angles: np.ndarray,
                        teeth_count: int) -> Tuple[float, float, List[int], np.ndarray]:
    # 筛选高阶分量
    high_order_components = [c for c in components if c.order >= teeth_count]
    
    if not high_order_components:
        return 0.0, 0.0, [], np.zeros_like(angles)
    
    # 高阶波数列表
    high_order_waves = [c.order for c in high_order_components]
    
    # 总振幅
    total_amplitude = sum(c.amplitude for c in high_order_components)
    
    # 重构信号
    angles_rad = np.radians(angles)
    reconstructed = np.zeros_like(angles, dtype=float)
    
    for c in high_order_components:
        reconstructed += c.coefficient_a * np.cos(c.order * angles_rad) + \
                        c.coefficient_b * np.sin(c.order * angles_rad)
    
    # RMS
    rms = np.sqrt(np.mean(reconstructed ** 2))
    
    return total_amplitude, rms, high_order_waves, reconstructed
```
