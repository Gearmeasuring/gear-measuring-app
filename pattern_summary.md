# 两个齿轮样本的规律分析

## 1. 样本参数对比

| 参数 | 样本1 (263751-018-WAV) | 样本2 (004-xiaoxiao1) |
|------|------------------------|----------------------|
| 齿数 ZE | 87 | 26 |
| 模数 m | 1.859 | 1.44 |
| 螺旋角 β | 25.3° | -25.0° |
| 分度圆直径 D₀ | 161.733 mm | 37.440 mm |
| 评估宽度 | 37.80 mm | 28.00 mm |
| **单齿Δφ范围** | **6.33°** | **19.98°** |

## 2. 关键发现

### 2.1 Δφ范围差异
- 样本2的Δφ范围（19.98°）是样本1（6.33°）的**3.15倍**
- 这导致样本2的齿向数据在角度域上覆盖更广

### 2.2 算法表现对比

**样本1 (ZE=87, 小Δφ):**
- 齿形：87阶次误差 < 10%（良好）
- 齿向：87阶次误差约48%，174阶次误差较小
- 整体表现：可接受

**样本2 (ZE=26, 大Δφ):**
- 齿形：26阶次误差5.3%（良好），但高阶次误差大
- 齿向：**26阶次误差944%**，52阶次误差219%
- 整体表现：**严重偏差**

### 2.3 问题规律

1. **齿向数据比齿形数据更难匹配**
2. **大螺旋角（大Δφ范围）导致更大误差**
3. **齿数少的齿轮误差更大**

## 3. 可能的原因

### 3.1 角度覆盖不均匀
- 大Δφ范围导致某些角度区域数据过密
- 合并后频谱出现假频（aliasing）

### 3.2 预处理方法不当
- 当前使用3阶多项式预处理
- 对于大螺旋角齿轮，可能需要不同阶数

### 3.3 频谱计算参数
- 当前使用最小二乘法计算振幅
- 可能需要针对大Δφ范围调整插值点数

## 4. 建议的优化方向

1. **自适应预处理阶数**
   - 根据螺旋角大小选择多项式阶数
   - 大螺旋角可能需要更高阶预处理

2. **角度域重采样**
   - 在合并曲线前进行均匀重采样
   - 避免角度覆盖不均匀导致的频谱失真

3. **频谱分析参数调整**
   - 根据齿数和Δφ范围调整FFT/最小二乘参数
   - 可能需要窗函数处理

4. **齿向专用算法**
   - 齿向和齿形可能需要不同的处理流程
   - 齿向数据受螺旋角影响更大
